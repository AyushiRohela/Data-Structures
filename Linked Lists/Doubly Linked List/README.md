# DOUBLY LINKED LIST

This folder consists of a file main.c with some of the functions possible on a doubly linked list.

## Functions implemented are:
1) CREATION
2) INSERTION
 - at head
 - at the end
 - after a given position
 - before a given position
3) DELETION
 - at head
 - at a given position
4) TRAVERSAL/PRINTING THE LIST
